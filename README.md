
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

Web App where you can write personal blog and publish articles.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
